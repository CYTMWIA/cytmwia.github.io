### 常用命令

更换 APT 源（USTC）
```bash
# 使 APT 支持 HTTPS
sudo apt-get install -y apt-transport-https ca-certificates
# 换源（Ubuntu 22）
curl https://mirrors.ustc.edu.cn/repogen/conf/ubuntu-https-4-jammy | sudo tee /etc/apt/sources.list
# 更新
sudo apt update
```

安装常用库（`opencv`, `boost`, `fmt`, `yaml-cpp`, `ceres-solver`）
```bash
sudo apt install -y libopencv-dev libboost-all-dev libfmt-dev libyaml-cpp-dev libceres-dev
```

### 连接 NUC

NUC 指代 部署视觉程序的电脑  
电脑 指代 我们自己的电脑/笔记本  

#### 准备工作

在 NUC 安装 DHCP 服务
```bash
sudo apt install isc-dhcp-server
```
编辑配置文件`/etc/dhcp/dhcpd.conf`
```
# /etc/dhcp/dhcpd.conf
default-lease-time 600;
max-lease-time 7200;
subnet 192.168.168.0 netmask 255.255.255.0 {
 range 192.168.168.200 192.168.168.250;
 option routers 192.168.168.1; # NUC IP，不同 NUC 对应不同 IP，避免 SSH 出错
}
```
编辑配置文件`/etc/default/isc-dhcp-server`
```
# /etc/default/isc-dhcp-server
INTERFACESv4=""
```
启用服务
```bash
sudo systemctl enable isc-dhcp-server.service
sudo systemctl restart isc-dhcp-server.service
```

#### 连接

使用网线直接连接电脑与NUC应该就可以通过IP连接彼此

#### Why Not ... ?

NUC和电脑使用同网段下的静态IP？  
这样确实可以连接彼此，但是需要手动修改Windows的网络设置，比较麻烦

使用路由器？  
那就不仅要准备网线，还需要准备个路由器

### 代码编辑

[Visual Studio Code](https://code.visualstudio.com/)  

插件（基础插件就不写了）
- [Remote - SSH](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-ssh)  
  远程编写代码  
  设置密钥登录可以省去输密码的麻烦  

- [clangd](https://marketplace.visualstudio.com/items?itemName=llvm-vs-code-extensions.vscode-clangd)  
  更强大的代码提示  
  在编写CMake项目时，可以在`CMakeLists.txt`中添加以下选项
  ```
  ## 为 clangd 导出编译选项
  set(CMAKE_EXPORT_COMPILE_COMMANDS ON)
  ```
  VSCode 配置 `clangd.arguments` 添加项（最好在工作区内修改）
  ```
  --compile-commands-dir=<compile_commands.json所在目录>
  ```
